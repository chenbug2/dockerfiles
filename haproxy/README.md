#### Custom usage:

    docker run \
        -d \
        --name haproxy \
        -p 443:443 \
        -v /your/path/haproxy.cfg:/etc/haproxy/haproxy.cfg \
        gists/haproxy

#### Compose example:

    haproxy:
      image: gists/haproxy
      container_name: nfsd
      ports:
        - "443:443"
      volumes:
        - /your/path/haproxy.cfg:/etc/haproxy/haproxy.cfg
      restart: always
